# A Visual Circuit Editor for Manim

It finally happened: I have nearly completed my visual circuit editor for the Manim library.

Many of you are probably familiar with Manim. It is a powerful and very convenient tool for creating clear technical animations. However, when it comes to working with electrical circuits, things can quickly become more complicated. Writing circuit descriptions by hand in Python takes a lot of time, especially when the schematic includes many components and connections.

That is why I created a visual web-based circuit editor for my own workflow. With this tool, you can draw a circuit directly in the browser and then export a ready-to-use Python description of the schematic for your Manim animation.

The editor is already fully functional, and I have used it myself to create several videos for the HighSNR channel. Now I am making it available as a free web tool for anyone who wants to use it.

One important note: to use the exported code correctly, make sure to download the companion Python class library as well. It contains the definitions for all the components available in the editor.

I hope this tool will be useful for your own animations, experiments, and technical videos.

Give it a try, and let me know what you think. I’ll be glad to hear your feedback, suggestions, and ideas for future improvements.
